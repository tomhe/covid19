COVID-19 Visualizations

Made by @tomhe with data from Johns Hopkins CSSE and inspired by Financial Times' Coronavirus tracker by John Burn-Murdoch et al.

Hint: The countries in the "Country" legends below the charts can be "shift clicked" to select only a few countries (or a single country).

Cumulative Deaths by Date

This chart shows the cumulative number of deaths by date.

Cumulative Deaths by Same Day of Outbreak

This chart shows the cumulative number of deaths by number of days since 10th death.

The lines for each country are the same as in the chart directly above, but aligned horizontally to roughly match the point in time when the outbreak reached 10 accumulated deaths.

Deaths per Day by Date

This chart shows the number deaths per day (7-day rolling average) by date.

Deaths per Day by Same Day of Outbreak

This chart shows the number deaths per day (7-day rolling average) by number of days since 3 deaths per day.

The lines for each country are the same as in the chart directly above, but aligned horizontally to roughly match the point in time when the outbreak reached 3 deaths per day (7-day rolling average).
